# WorkSync

WorkSync is an Employee & Attendance System.

## Tech stack

- Frontend: Next.js (TypeScript)
- Backend: ASP.NET Core Web API (C#)
- Database: SQL Server (EF Core)
- Docker + Docker Compose
- CI/CD: GitHub Actions, Fly.io (backend), Vercel (frontend)
